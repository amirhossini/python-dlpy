Parameters

Kernel size of the convolution filters. Default: 3

n_filtersiter-of-ints, optional

List of numbers of filter in each convolution layers. Default: (16, 16)

stridesiter-of-ints, optional

List of stride in each convolution layers.

batch_norm_firstbool, optional
